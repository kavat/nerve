About

Welcome to NERVE, the Network Exploitation, Reconnaissance & Vulnerability Engine

Its purpose is to scan and identify short and long lived security flaws using one-time or continous scan modes on a given network or host.

This can be acheived either programmatically via the API or using the Graphical User Interface

How does it work?

  1. You configure your network or domain list e.g. 172.15.0.0/24
  2. You define which attacks to run (Brute Force, etc.)
  3. You run the assessment...
  4. You wait for the results!
